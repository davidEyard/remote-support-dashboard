# Remote Support Dashboard (React)

A mini IT help-desk style dashboard built with React.

## Features
- Ticket list with search + filters (status, priority)
- Ticket detail panel with steps taken
- Update ticket status (In Progress / Resolved)
- Knowledge base panel with quick-fix summaries
- SHIP button generates a summary report + downloads CSV/JSON

## Tech
- React (Create React App)

## Run locally
npm install
npm start
